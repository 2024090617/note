  - How to clarify a requirment.

    - A task is *atomic* for AI if:

    * It has **one clear input**
    * It has **one measurable output**
    * It can be **verified automatically**
    * It does **not require hidden context**
  - ### Atomicity Check Prompt (Key Trick)

  Ask the model:

  > “Can this task be completed by a single function call or code block, and verified automatically?”
  >

  If  **NO → break it down further** .
